Основные команды для работы с github



Основные команды консоли Git Bash 24
git status -узнать текущий статус репозитория
git log - узнать о всех коммитах
git remote -v - узнать какие origin
git branch - ветка
HEAD^^ или HEAD~2 перемещение указателя HEAD


git mkdir forExampleNameDir - создать папку, необходимо написать ее название
git init - Инициализировать git в папке, делается первый раз, создается скрытая папка .git


git add . - Подготовили файлы к фиксации
git commit -m - Фиксируем ранее подготовленные файлы
git commit -m "Add name commit" - комит, действие пишется в настоящем времени Add

git diff


git diff - Покажет ращницу между текущим НЕОТСЛЕЖИВАЕМЫМ состоянием репозитория и последним снимком репозитория.
git diff -- staged - Покажет ращницу между текущим ОТСЛЕЖИВАЕМЫМ состоянием репозитория и последним снимком репозитория.
git diff -- COMMIT_ID - Покажет ращницу между текущим состоянием репозитория и УКАЗАННЫМ снимком репозитория.

git reset


Команда используется для переписывания истории
git reset --soft возвращает проект к указанному коммиту, при этом переводит все коммиты после указанного в ОТСЛЕЖИВАЕМУЮ ЗОНУ
git reset -- mixed возвращает проект к указанному коммиту, при этом переводит все коммиты после указанного в НЕОТСЛЕЖИВАЕМУЮ зону
git reset -- hard врзвращает проект к указанному коммиту, при этом полностю улаояет все коммиты прсое указанного. УДАЛЯЕТ КОММИТЫ БЕЗВОЗВРАТНО!!!

git checkout


Используется для перемещения между коммитами, версиями отдельных фалов, ветками, если мы хотим посмотреть как выглядел проект в каком-то снимке в прошлом. Коммиты не удалятся. Все мзенения не сохраняются (Удаляются сборщиком мусора).
git checkout master - возврат на последнюю версию.
git checkout adslnb49uhva9etrovn34us03ffafn944b9494 -- file1 file2
